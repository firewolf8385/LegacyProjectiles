# LegacyProjectiles &middot; [![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-square)](http://makeapullrequest.com) [![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg?style=flat-square)](https://github.com/JadedMC/ProjectileKB/blob/master/LICENSE)

LegacyProjectiles is a simple plugin that removes the player's velocity from launched projectiles, like they used to be prior to the combat update 1.9. No configuration, no commands, just plug and play!

</p>

## Requirements
* Java 17 or newer
* Server Software with Bukkit API.
